Mengenal Variabel

Definisi

Variabel adalah sebuah wadah yang bisa menyimpan nilai.

Membuat Variabel

Variabel pada python dituliskan sebagai berikut

x = 10
Penjelasan
  • x adalah nama variabel
  • = adalah assignment atau penugasan
  • 10 adalah nilai yang akan disimpan kedalam variabel x

Pada dasarnya nama variabel pada python dituliskan dengan huruf, tidak boleh menggunakan spasi (bisa gunakan underscore), dan tidak boleh diawali dengan angka.


    nilai_y = 15
    juta10 = 10000000
    nilaiZ = 17.5
Memanggil Variabel

Setelah variabel dibuat kita bisa memanggil variabel tersebut, misalnya kita ingin menampilkan variabel tersebut, cara pemanggilannya adalah sebagai berikut

print(x)

Selanjutnya kita bisa menambahkan teks untuk ditampilkan bersama variabel

print("Nilai x = ", x)

Input Data Dari User

Definisi

Input pengguna digunakan untuk membuat program dapat meminta pengguna untuk memasukkan teks atau nilai, dan kemudian menggunakan input tersebut dalam eksekusi program. Data yang diinput akan mengembalikan nilai string.

Input User pada Python

Untuk mendapatkan input dari pengguna dalam bahasa pemrograman Python, Anda dapat menggunakan fungsi input(). Berikut adalah contoh sederhana:


    # Meminta input dari pengguna
    nama_pengguna = input("Masukkan nama Anda: ")

    # Menampilkan hasil input
    print("Halo, " + nama_pengguna + "! Selamat datang.")

Perlu diingat bahwa fungsi input() selalu mengembalikan data dalam bentuk string. Jika Anda perlu melakukan operasi matematika atau pemrosesan data numerik lainnya, pastikan untuk mengonversi input ke tipe data yang sesuai. Misalnya, menggunakan int() untuk mengonversi input menjadi bilangan bulat.

    # Mengambil input dari user

Operasi Aritmatika

Definisi

Operasi aritmatika adalah operasi matematika dasar yang dapat dilakukan pada bilangan untuk melakukan perhitungan. Dalam Python, Anda dapat melakukan operasi aritmatika seperti penambahan, pengurangan, perkalian, pembagian, dan lainnya.

Operasi Aritmatika

Berikut adalah contoh operasi aritmatika dalam Python:

baca selengkapnya

    # Operasi aritmatika

    a = 10
    b = 3

    # operasi tambah + 
    hasil = a + b
    print(a, '+', b, '=', hasil)

    # operasi pengurangan - 
    hasil = a - b
    print(a, '-', b, '=', hasil)

    # operasi perkalian * 
    hasil = a * b
    print(a, '*', b, '=', hasil)

    # operasi pembagian / 
    hasil = a / b
    print(a, '/', b, '=', hasil)

    # operasi eksponen (pangkat) ** 
    hasil = a ** b
    print(a, '**', b, '=', hasil)

    # operasi modulus (sisa bagi) % 
    hasil = a % b
    print(a, '%', b, '=', hasil)

    # operasi floor division (lawan dari modulus, sama dengan pembagian namun dibulatkan kebawah) // 
    hasil = a // b
    print(a, '//', b, '=', hasil)

Operasi Komparasi

Definisi

Operasi komparasi adalah operasi yang membandingkan dua nilai atau ekspresi untuk menentukan hubungan antara keduanya. Hasil dari operasi komparasi adalah nilai kebenaran, yaitu True (benar) atau False (salah).

Operasi Komparasi

Dalam Python, terdapat beberapa operator komparasi yang umum digunakan:

1. Sama dengan (==)

Membandingkan apakah nilai x sama dengan nilai y

baca selengkapnya

    x = 5
    y = 5
    hasil = x == y
    print(hasil)  # Output: True
2. Tidak sama dengan (!=)

Membandingkan apakah nilai a tidak sama dengan nilai b

baca selengkapnya

    a = 10
    b = 7
    hasil = a != b
    print(hasil)  # Output: True
3. Kurang dari (<)

Membandingkan apakah nilai m kurang dari n

baca selengkapnya

    m = 8
    n = 12
    hasil = m < n
    print(hasil)  # Output: True
4. kurang dari sama dengan (<=)

Membandingkan apakah nilai p kurang dari sama dengan nilai q

baca selengkapnya

    p = 10
    q = 10
    hasil = p <= q
    print(hasil)  # Output: True
5. Lebih dari (>)

Membandingkan apakah nilai i lebih dari nilai j

baca selengkapnya

    i = 15
    j = 10
    hasil = i > j
    print(hasil)  # Output: True
6. Lebih dari sama dengan (>=)

Membandingkan apakah nilai c lebih dari sama dengan nilai d

baca selengkapnya

    c = 7
    d = 7
    hasil = c >= d
    print(hasil)  # Output: True

Operasi Logika atau Boolean

Definisi

Operasi logika atau Boolean adalah operasi matematika yang bekerja pada nilai True atau False. Dalam pemrograman, operasi logika sering digunakan untuk membuat keputusan berdasarkan kondisi yang diberikan.

Operator Logika

Beberapa operator logika utama yang digunakan dalam Python adalah:

1. Operator `and`

Menghasilkan `true` jika kedua kondisi bernilai `true`.

baca selengkapnya
                                                    
    x = True
    y = False
    hasil = x and y
    print(hasil)  # Output: False
2. Operator `or`

Menghasilkan `true` jika salah satu kondisi atau semua kondisi bernilai `true`.

baca selengkapnya
                                                    
    a = True
    b = False
    hasil = a or b
    print(hasil)  # Output: True
3. Operator `not`

Menghasilkan kebalikan dari nilai kondisi atau ekspresi.

baca selengkapnya
                                                    
    m = True
    hasil = not m
    print(hasil)  # Output: False
4. Operator `xor` (exclusive or)

Menghasilkan `True` jika tepat satu dari dua kondisi atau ekspresi bernilai `True`.

baca selengkapnya
                                                    
    p = True
    q = False
    hasil = p ^ q
    print(hasil)  # Output: True

Operator Bitwise

Definisi

Operator bitwise adalah operator yang bekerja pada level bit dari representasi biner dari nilai-nilai dalam suatu operasi.

Operator Bitwise

Dalam Python, terdapat beberapa operator bitwise yang digunakan untuk melakukan operasi pada bit:

1. AND Bitwise (&)

Menetapkan bit ke-i menjadi 1 jika kedua bit pada posisi ke-i dari operand adalah 1.

baca selengkapnya
                                                    
    a = 5  # Binernya: 0b0101
    b = 3  # Binernya: 0b0011
    hasil_and = a & b
    print(hasil_and)  # Output: 1 (Binernya: 0b0001)
2. OR Bitwise (|)

Menetapkan bit ke-i menjadi 1 jika salah satu atau kedua bit pada posisi ke-i dari operand adalah 1.

baca selengkapnya
                                                    
    x = 5  # Binernya: 0b0101
    y = 3  # Binernya: 0b0011
    hasil_or = x | y
    print(hasil_or)  # Output: 7 (Binernya: 0b0111)
3. XOR Bitwise (^)

Menetapkan bit ke-i menjadi 1 jika hanya satu dari dua bit pada posisi ke-i dari operand adalah 1.

baca selengkapnya
                                                    
    p = 5  # Binernya: 0b0101
    q = 3  # Binernya: 0b0011
    hasil_xor = p ^ q
    print(hasil_xor)  # Output: 6 (Binernya: 0b0110)
4. NOT Bitwise (~)

Menyandi ulang setiap bit, mengubah 0 menjadi 1 dan 1 menjadi 0.

baca selengkapnya
                                                    
    m = 5  # Binernya: 0b0101
    hasil_not = ~m
    print(hasil_not)  # Output: -6 (Binernya: -0b0110)
5. Shift Left (<<)

Memindahkan semua bit operand ke kiri sejumlah langkah tertentu.

baca selengkapnya
                                                    
    c = 4  # Binernya: 0b0100
    hasil_shift_left = c << 1
    print(hasil_shift_left)  # Output: 8 (Binernya: 0b1000)
6. Shift Right (>>)

Memindahkan semua bit operand ke kanan sejumlah langkah tertentu.

baca selengkapnya
                                                    
    d = 8  # Binernya: 0b1000
    hasil_shift_right = d >> 1
    print(hasil_shift_right)  # Output: 4 (Binernya: 0b0100)

IF dan ELSE Statement

Definisi

Dalam bahasa pemrograman Python, IF dan ELSE adalah pernyataan kontrol alur program yang digunakan untuk membuat keputusan berdasarkan kondisi tertentu. Pernyataan ini memungkinkan Anda menjalankan blok kode tertentu jika kondisi tertentu terpenuhi.

IF dan ELSE Statement

Berikut adalah contoh penggunaan if dan else:

1. Pernyataan IF:
baca selengkapnya
                                                    
    nilai = 75

    if nilai >= 70:
        print("Lulus")

Dalam contoh di atas, blok kode di bawah pernyataan IF akan dieksekusi hanya jika kondisinya (nilai >= 70) benar. Jika kondisi itu benar, pesan "Lulus" akan dicetak.

2. Pernyataan IF dan ELSE:
baca selengkapnya
                                                    
    nilai = 55

    if nilai >= 70:
        print("Lulus")
    else:
        print("Tidak Lulus")

Dalam contoh ini, jika kondisi di bawah pernyataan IF (nilai >= 70) benar, blok kode di bawah IF akan dieksekusi. Jika kondisinya salah, blok kode di bawah pernyataan ELSE akan dieksekusi.

3. Pernyataan IF, ELIF, dan ELSE:
baca selengkapnya
                                                    
    nilai = 60

    if nilai >= 70:
        print("Lulus")
    elif nilai >= 50:
        print("Cukup")
    else:
        print("Tidak Lulus")

Dalam contoh ini, ELIF (singkatan dari "ELSE IF") digunakan untuk menambahkan kondisi tambahan. Program akan mengevaluasi kondisi secara berurutan, dan blok pertama yang memenuhi kondisi akan dieksekusi. Jika tidak ada yang memenuhi, blok kode di bawah pernyataan ELSE akan dieksekusi.

Definisi

    2

    4

Perlu diperhatikan bahwa perluasan atau pemutusan loop harus dikelola secara hati-hati agar loop tidak menjadi tak terbatas (infinite loop), yaitu loop yang tidak pernah berhenti. Oleh karena itu, pastikan kondisi dalam while loop dapat mencapai keadaan salah (False) pada suatu saat.

Continue dan Pass

Definisi

continue, pass, dan break adalah pernyataan khusus yang digunakan dalam struktur kontrol alur di Python, seperti dalam loop (for dan while) atau dalam blok kondisional (if, elif, dan else).

Continue dan Pass

Berikut adalah penjelasan dan contoh penggunaan dari statement continue, pass, dan break

1. continue
  • Pernyataan continue digunakan untuk menghentikan iterasi saat ini dalam loop dan melanjutkan ke iterasi berikutnya.
  • Ketika continue dieksekusi, blok kode di bawahnya dalam loop tidak dijalankan, dan iterasi berikutnya dimulai.
baca selengkapnya
Contoh penggunaan continue dalam loop:
                                                    
    for i in range(5):
        if i == 2:
            continue
        print(i)
Output:
                                                    
    0
    1
    3
    4

Pada contoh di atas, ketika i sama dengan 2, pernyataan continue menyebabkan iterasi tersebut dilewati, sehingga print(i) tidak dijalankan.

2. pass
  • Pernyataan pass digunakan sebagai pernyataan dummy atau tempat holder saat Anda tidak ingin menulis kode di dalam suatu blok.
  • pass tidak melakukan apa-apa dan tidak mengubah alur program.
baca selengkapnya
Contoh penggunaan pass dalam loop:
                                                    
    for i in range(5):
        if i == 2:
            pass
        else:
            print(i)
Output:
                                                    
    0
    1
    3
    4

Pada contoh di atas, saat i sama dengan 2, pass membuat blok else tetap kosong, dan eksekusi melanjutkan ke iterasi berikutnya.

3. break
  • Pernyataan break digunakan untuk menghentikan loop secara keseluruhan ketika suatu kondisi tertentu terpenuhi.
  • Setelah break dieksekusi, eksekusi keluar dari loop.
baca selengkapnya
Contoh penggunaan break dalam loop:
                                                    
    for i in range(5):
        if i == 3:
            break
        print(i)
Output:
                                                    
    0
    1
    2

Pada contoh di atas, ketika i sama dengan 3, pernyataan break menyebabkan loop berhenti, dan eksekusi keluar dari loop.

Deep Copy List

Definisi

Dalam Python, deep copy list digunakan untuk membuat salinan independen dari list, termasuk semua elemen list dan elemen-elemen yang bersarang di dalamnya. Artinya, jika list yang asli berisi list bersarang (nested list), deep copy juga akan menciptakan salinan baru dari list bersarang tersebut.

Deep Copy List

Untuk melakukan deep copy, kita dapat menggunakan modul copy yang menyediakan fungsi deepcopy. Berikut adalah contoh penggunaannya:

baca selengkapnya

    # Membuat list bersarang

    # Menampilkan hasil deep copy dan list asli
    print(original_list)        # Output: [1, [2, 3], 4]
    print(deep_copied_list)     # Output: [1, [2, 3], 4]

Penting untuk dicatat bahwa deep copy menghasilkan salinan yang sepenuhnya independen dari list asli. Sehingga, ketika kita mengubah elemen-elemen di dalam deep copy, list asli tidak terpengaruh.

Enumerate:

Tuples dan Set

Definisi

Tuple adalah struktur data di Python yang bersifat immutable, artinya elemennya tidak dapat diubah setelah tuple dibuat, dan didefinisikan dengan menggunakan tanda kurung biasa (). Set adalah struktur data di Python yang menyimpan elemen-elemen unik tanpa urutan tertentu, dan didefinisikan dengan menggunakan tanda kurung kurawal {}.

Tuples dan Set

Berikut adalah sedikit penjelasan mengenai struktur data Tuples dan Set

1. Tuples

Tuple adalah struktur data yang mirip dengan list, namun bersifat immutable (tidak dapat diubah). Artinya valuenya tidak dapat ditambahkan, dihapus, atau diubah. Tuple didefinisikan dengan menggunakan tanda kurung biasa ().

baca selengkapnya
                                                    
    my_tuple = (1, 2, 'tiga', 4.0)
    print(my_tuple)  # Output: (1, 2, 'tiga', 4.0)

Anda dapat mengakses elemen-elemen tuple menggunakan indeks, sama seperti list. Namun, karena tuple bersifat immutable, Anda tidak dapat mengubah nilai elemen dalam tuple setelah tuple tersebut dibuat.

2. Set

Set adalah struktur data yang digunakan untuk menyimpan kumpulan elemen unik tanpa adanya urutan tertentu. Set tidak mengizinkan adanya elemen duplikat. Set didefinisikan dengan menggunakan tanda kurung kurawal {}.

baca selengkapnya
                                                    
    my_set = {1, 2, 3, 4, 5}

    # Gabungan set
    set1 = {1, 2, 3}
    set2 = {3, 4, 5}
    union_set = set1.union(set2)
    print(union_set)  # Output: {1, 2, 3, 4, 5}

Set sangat berguna ketika Anda perlu menyimpan koleksi elemen unik dan melakukan operasi matematika himpunan pada mereka. Namun, perlu diingat bahwa set bersifat tidak berurutan, jadi Anda tidak dapat mengandalkan urutan elemen dalam set.

Pengenalan Fungsi

Definisi

Fungsi adalah blok (kumpulan) kode yang dirancang untuk menjalankan tugas tertentu (spesifik) dan dapat dipanggil dari bagian lain dalam program setelah fungsi tersebut dibuat.

Pengenalan Fungsi

Berikut adalah beberapa hal penting yang harus diketahui dalam fungsi Python:

1. Deklarasi Fungsi:

Fungsi dideklarasikan menggunakan kata kunci def, diikuti oleh nama fungsi dan parameter dalam tanda kurung. Blok kode fungsi diidentasi di bawah deklarasi.

baca selengkapnya
                                                    
    def contoh_fungsi(parameter1, parameter2):
        # Blok kode fungsi
        hasil = parameter1 + parameter2
        return hasil
2. Memanggil Fungsi

Fungsi dipanggil dengan menyebutkan nama fungsi dan memberikan nilai argumen sesuai dengan parameter yang dideklarasikan.

baca selengkapnya
                                                    
    hasil = contoh_fungsi(10, 20)
    print(hasil)  # Output: 30
3. Parameter dan Argumen

Parameter adalah Variabel yang digunakan dalam deklarasi fungsi. Sedangkan Argumen adalah Nilai yang diberikan saat memanggil fungsi.

baca selengkapnya
                                                    
    def tambah(a, b):
        return a + b
    
    hasil = tambah(3, 4)
    print(hasil)  # Output: 7

Dari kode diatas, (a, b) adalah parameter, dan (3, 4) adalah argumen.

4. Nilai Kembalian

Fungsi dapat mengembalikan nilai menggunakan kata kunci return. Nilai ini dapat digunakan atau disimpan saat memanggil fungsi.

baca selengkapnya
                                                    
    def kuadrat(x):
        return x ** 2
    
    hasil_kuadrat = kuadrat(5)
    print(hasil_kuadrat)  # Output: 25

Jika tidak ada pernyataan return, fungsi secara implisit mengembalikan None.

5. Nilai Default Parameter

Parameter dalam fungsi dapat memiliki nilai default yang digunakan jika nilai tidak diberikan saat memanggil fungsi.

baca selengkapnya
                                                    
    def pangkat(base, eksponen=2):
        return base ** eksponen
    
    hasil_default = pangkat(3)
    hasil_khusus = pangkat(3, 4)
    
    print(hasil_default)  # Output: 9
    print(hasil_khusus)   # Output: 81

Dengan menggunakan default parameter, saat pemanggilan fungsi, jika argumen 'eksponen' tidak diisi, maka fungsi akan menggunakan nilai default parameter yang sudah didefinisikan saat deklarasi fungsi.

6. Fungsi Lambda

Fungsi lambda adalah fungsi kecil yang didefinisikan tanpa menggunakan kata kunci def. Mereka sering digunakan untuk fungsi sederhana.

baca selengkapnya
                                                    
    kali_dua = lambda x: x * 2
    hasil = kali_dua(4)
    print(hasil)  # Output: 8

Fungsi rekursif adalah fungsi yang memanggil dirinya sendiri.

baca selengkapnya

Fungsi Dengan kwargs

Definisi

Fungsi dengan kwargs (singkatan dari "keyword arguments") adalah salah satu fitur yang memungkinkan Anda untuk mengirimkan argumen ke dalam fungsi Python menggunakan nama parameter dan nilai yang sesuai. kwargs memungkinkan Anda untuk menangani argumen fungsi dengan cara yang lebih fleksibel, karena Anda dapat mengirimkannya dalam bentuk pasangan kunci-nilai, dan fungsi akan menerimanya sebagai kamus (dictionary).

Fungsi Dengan kwargs

Berikut adalah beberapa langkah untuk menjelaskan cara menggunakan kwargs dalam fungsi Python:

1. Mendefinisikan Fungsi:

Gunakan dua tanda bintang (**) diikuti oleh nama parameter yang ingin Anda gunakan (biasanya menggunakan kwargs, atau sesuai keinginan).

baca selengkapnya
                                                    
    def contoh_fungsi(**kwargs):
        # Isi fungsi akan menggunakan kwargs
        for k, v in kwargs.items():
            print(f"{k}: {v}")
2. Memanggil Fungsi dengan kwargs:

Saat memanggil fungsi, Anda dapat menuliskan argumen dalam bentuk pasangan kunci-nilai. Ini akan dikumpulkan oleh parameter kwargs dalam bentuk dictionary.

baca selengkapnya
                                                    
    contoh_fungsi(nama="John", umur=25, pekerjaan="Programmer")

Dalam contoh ini, kita memanggil fungsi contoh_fungsi dengan tiga pasangan kunci-nilai yang berbeda.

3. Mengakses Nilai di Dalam Fungsi:

Dalam tubuh fungsi, kwargs akan dianggap sebagai dictionary. Oleh karena itu, Anda dapat mengakses nilai-nilai tersebut menggunakan kunci.

baca selengkapnya
                                                    
    def contoh_fungsi(**kwargs):
        for k, v in kwargs.items():
            print(f"{k}: {v}")

    contoh_fungsi(nama="John", umur=25, pekerjaan="Programmer")

    nama: John
    umur: 25
    pekerjaan: Programmer

Global dan Local Scope

Definisi

Dalam Python, konsep tentang "scope" (lingkup) merujuk pada tempat di mana suatu variabel dapat diakses atau dimodifikasi. Terdapat dua jenis scope utama dalam Python: global scope dan local scope.

Global dan Local Scope

Ada dua jenis scope utama dalam Python, yaitu global dan local, berikut penjelasannya:

1. Global Scope

Adalah variabel yang didefinisikan di luar fungsi atau blok kode utama.

baca selengkapnya
  • Definisi: Variabel yang didefinisikan di luar semua fungsi atau blok kode disebut variabel dengan global scope.
  • Aksesibilitas: Variabel dengan global scope dapat diakses dari mana saja dalam program, baik di dalam fungsi maupun di luar fungsi.
  • Cara Mendefinisikan: Variabel didefinisikan di luar fungsi atau blok kode utama.
Contoh:
                                                    
    global_variable = 10

    def my_function():
        print(global_variable)

    my_function()
    print(global_variable)

variabel "global_variable" didefinisikan diluar fungsi, dia bisa digunakan di semua bagian program.

2. Local Scope

Adalah variabel yang didefinisikan di dalam fungsi atau blok kode tertentu. Tidak termasuk perulangan dan percabangan.

baca selengkapnya
  • Definisi: Variabel yang didefinisikan di dalam sebuah fungsi disebut variabel dengan local scope. Local scope juga berlaku untuk variabel yang didefinisikan dalam blok percabangan atau perulangan.
  • Aksesibilitas: Variabel dengan local scope hanya dapat diakses di dalam fungsi atau blok kode tempat variabel tersebut didefinisikan.
  • Cara Mendefinisikan: Variabel didefinisikan di dalam fungsi atau blok kode tertentu.
Contoh:
                                                    
    def my_function():
        local_variable = 5
        print(local_variable)

    my_function()

    # Akses variabel local di luar fungsi akan menghasilkan NameError
    # print(local_variable)

variabel "local_variable" didefinisikan di dalam fungsi, dia hanya akan berfungsi di dalam fungsi tempat dia didefinisikan.

Global Keyword

Jika kita ingin membuat variabel local scope bisa diakses diluar fungsi maka kita bisa menggunakan keyword "global" yang dituliskan sebelum nama variabelnya, berikut langkah-langkahnya.

baca selengkapnya
  • Definisikan terlebih dahulu variabel global scope-nya.
  • Buat fungsi dan tuliskan didalamnya keyword "global" diikuti dengan nama variabel local scope, gunakan nama yang sama dengan nama variabel global scope.
  • Dengan keyword "global", itu menandakan variabel tersebut merujuk pada variabel global scope.
  • Variabel tersebut sudah bisa dioperasikan didalam fungsi maupun luar fungsi.
Contoh:
                                                    
    global_variable = 10

    def my_function():
        global global_variable
        global_variable += 5
        print(global_variable)
    
    my_function()
    print(global_variable)

penggunaan variabel dengan global scope harus dilakukan dengan hati-hati untuk menghindari konflik dalam program Python.

Import Statement

Definisi
  • Import Statement adalah mekanisme dalam Python yang memungkinkan kita untuk menggunakan kode yang telah ditulis di file atau modul terpisah.
  • Dengan menggunakan import, kita dapat mengakses definisi variabel, fungsi, atau kelas yang ada di dalam modul atau file eksternal dan menggunakan mereka dalam kode kita.
Import Statement

Ada beberapa cara untuk menggunakan import statement di Python. Beberapa contoh umum melibatkan penggunaan kata kunci seperti import, from, dan as.

1. Import Modul Keseluruhan:
baca selengkapnya
- cara penggunaan
                                                    
    import math
- contoh penggunaan
                                                    
    # Mengimport modul math
    import math

    # Menggunakan fungsi sqrt dari modul math
    result = math.sqrt(25)
    print(result)  # Output: 5.0
2. Import Spesifik Dari Modul:
baca selengkapnya
- cara penggunaan
                                                    
    from math import sqrt
- contoh penggunaan
                                                    
    # Mengimport hanya fungsi sqrt dari modul math
    from math import sqrt

    # Menggunakan fungsi sqrt tanpa menyertakan nama modul
    result = sqrt(25)
    print(result)  # Output: 5.0
3. Memberikan Alias:
baca selengkapnya
- cara penggunaan
                                                    
    import math as m
- contoh penggunaan
                                                    
    # Mengimport modul math dengan alias 'm'
    import math as m

    # Menggunakan fungsi sqrt dari modul math menggunakan alias
    result = m.sqrt(25)
    print(result)  # Output: 5.0
Catatan
Penting untuk Diperhatikan:
  • Import statement memungkinkan modularitas dan pengorganisasian kode yang lebih baik, memisahkan fungsionalitas ke dalam modul-modul terpisah.
  • Penggunaan import yang baik dapat membantu mencegah duplikasi kode dan membuat kode lebih mudah dipelihara.
  • Modul-modul Python yang sudah ada (seperti math, os, dll.) adalah bagian integral dari bahasa Python dan dapat diimpor untuk digunakan di berbagai aplikasi.

Membuat Module

Definisi

Sebuah modul dalam Python adalah file yang berisi kode Python. Modul dapat berisi definisi variabel, fungsi, dan kelas yang dapat digunakan di dalam program Python lainnya.

Fungsi Module
  • Modul memungkinkan pemisahan kode berdasarkan fungsionalitas. Dengan memisahkan kode ke dalam modul, kita dapat membuat struktur yang lebih terorganisir dan mudah dimengerti.
  • Modul memungkinkan kode yang telah ditulis untuk digunakan kembali di berbagai bagian dari program atau di berbagai program.
  • Dengan memisahkan fungsionalitas ke dalam modul-modul terpisah, kode menjadi lebih mudah dibaca dan dipelihara.
Membuat Module

Berikut beberapa hal yang perlu diketahui tentang module.

1. Membuat dan Menggunakan Modul:
baca selengkapnya

Untuk membuat modul, kita cukup membuat file Python yang berisi definisi-definisi kita (variabel, fungsi, atau kelas).

                                                    
    # Contoh modul dalam file my_module.py

    def hello_world():
        print("Hello, World!")

    def square(x):
        return x ** 2

Modul tersebut dapat digunakan di program Python lain dengan menggunakan pernyataan import.

        
    # Contoh penggunaan modul

    import my_module

    my_module.hello_world()  # Memanggil fungsi dari modul
    result = my_module.square(5)  # Menggunakan fungsi lain dari modul
    print(result)
2. Namespace Module
baca selengkapnya

Ketika kita mengimpor modul, Python menciptakan sebuah namespace untuk modul tersebut. Ini berarti kita dapat mengakses definisi-modul menggunakan sintaks nama_modul.nama_definisi.

                                                    
    # Contoh namespace modul

    import my_module

    my_module.hello_world()  # Memanggil fungsi dari modul
3. __name__ dan __main__
baca selengkapnya

Modul juga memiliki atribut khusus __name__. Ketika sebuah modul diimpor, __name__ akan set ke nama modul. Namun, jika sebuah modul dijalankan sebagai program utama, __name__ akan diatur menjadi "__main__". Ini memungkinkan kita untuk mengeksekusi kode tertentu hanya ketika modul dijalankan sebagai program utama.

                                                    
    # Contoh penggunaan __name__

    def my_function():
        print("This is my function.")

    if __name__ == "__main__":
        print("This is the main program.")
        my_function()

Dengan menggunakan __name__, kita dapat memisahkan kode yang hanya ingin dijalankan ketika modul dijalankan sebagai program utama.

Itulah beberapa aspek penting terkait dengan modul dalam Python. Dengan memahami konsep ini, kita dapat membuat kode yang lebih terorganisir, mudah dimengerti, dan dapat digunakan kembali.

Membuat Package

Definisi
  • Package adalah cara untuk mengorganisir dan mengelompokkan modul-modul terkait ke dalam struktur direktori.
  • Ini membantu dalam pengorganisasian proyek-proyek Python yang lebih besar dan memisahkan fungsionalitas ke dalam unit-unit terpisah.
  • Package juga memberikan cara untuk menghindari konflik nama antar-modul dan meningkatkan keterbacaan kode.
  • Dengan menggunakan package, pengembang dapat membuat proyek yang lebih terorganisir, mudah diurus, dan mudah dipahami.
Membuat Package

Berikut beberapa hal yang perlu diketahui tantang package.

1. Struktur Package:
baca selengkapnya

Untuk membuat package, Anda harus memiliki setidaknya satu direktori yang berisi file khusus bernama __init__.py. Direktori ini akan dianggap sebagai package oleh Python. File __init__.py dapat berisi kode yang dieksekusi saat package diimpor. Selain itu, direktori tersebut dapat berisi modul-modul Python atau sub-package.

2. Contoh Struktur Package:
baca selengkapnya

Misalnya, kita ingin membuat package bernama my_package dengan dua modul di dalamnya. Struktur direktori dan file bisa terlihat seperti ini:

                                                    
    my_package/
    |-- __init__.py
    |-- module1.py
    |-- module2.py
3. Cara Mengimpor Modul dari Package:
baca selengkapnya

Setelah kita memiliki package, kita dapat mengimpor modul-modul di dalamnya menggunakan pernyataan import. Contoh:

                                                    
    # Mengimpor modul1 dari my_package
    from my_package import module1

    # Menggunakan fungsi dari modul1
    module1.some_function()

Atau kita bisa mengimpor seluruh package:


    # Mengimpor seluruh my_package
    import my_package

    # Menggunakan fungsi dari modul1
    my_package.module1.some_function()
4. Sub-Package:
baca selengkapnya

Package juga dapat berisi sub-package, yang merupakan package di dalam package. Struktur seperti ini memberikan tingkat organisasi yang lebih tinggi dan membantu dalam memecah proyek menjadi bagian-bagian yang lebih kecil dan terkelola.
Contoh Struktur dengan Sub-Package:

                                                    
    my_project/
    |-- __init__.py
    |-- module1.py
    |-- package1/
    |   |-- __init__.py
    |   |-- module2.py
    |-- package2/
        |-- __init__.py
        |-- module3.py

Pastikan bahwa setiap direktori dalam struktur package memiliki file __init__.py, bahkan jika file tersebut kosong. File ini memberi tahu Python bahwa direktori tersebut adalah bagian dari package.

Init Pada Package

Definisi
  • Di Python, file __init__.py adalah sebuah file khusus yang digunakan untuk menandai suatu direktori sebagai sebuah paket (package).
  • File ini bisa kosong atau berisi kode Python, dan keberadaannya memberi tahu Python bahwa direktori tersebut adalah bagian dari suatu paket.
Init Pada Package

Berikut adalah penjelasan secara lengkap beserta contoh penggunaan __init__.py pada package:

baca selengkapnya
1. Menandai Direktori Sebagai Package:
  • Ketika Python menemui sebuah direktori yang berisi file __init__.py, itu akan mengenali direktori tersebut sebagai sebuah package.
2. Inisialisasi Package:
  • File __init__.py dapat berisi kode yang akan dieksekusi saat package diimpor. Ini memungkinkan Anda untuk melakukan inisialisasi atau menentukan variabel-variabel yang dapat diakses dari dalam package tersebut.
3. Contoh Penggunaan:

  • Misalkan kita memiliki struktur direktori sebagai berikut:

    
    mypackage/
    ├── __init__.py
    ├── module1.py
    ├── module2.py

  • Berikut adalah isi dari file __init__.py:

    
    # mypackage/__init__.py

    print("Initializing mypackage")

    # Variabel yang dapat diakses dari dalam package
    package_variable = "Hello from mypackage"

  • Selanjutnya, Anda dapat mengimpor package dan mengakses variabel dari file __init__.py:

    
    # main.py

    import mypackage

    # Mencetak pesan inisialisasi dari __init__.py
    print(mypackage.package_variable)

    Ketika menjalankan main.py, outputnya akan menjadi:

    
    Initializing mypackage
    Hello from mypackage

  • Dengan demikian, file __init__.py memberikan kesempatan untuk mengeksekusi kode saat package diimpor.

4. Versi Tanpa __init__.py:
  • Jika direktori package tidak memiliki file __init__.py, Python mungkin tidak mengenali direktori tersebut sebagai package, dan beberapa fitur package mungkin tidak berfungsi sebagaimana mestinya.
5. Catatan:
  • Seiring dengan pengenalan Python 3.3, __init__.py tidak selalu diperlukan untuk package, tetapi untuk kompatibilitas mundur dan kejelasan, disarankan untuk tetap menyertakan file tersebut.

Standard Library

Definisi
  • Standard Library pada Python adalah kumpulan modul dan paket (libraries) bawaan yang disertakan dalam instalasi Python.
  • Modul-modul ini menyediakan berbagai fungsi dan alat untuk membantu pengembangan perangkat lunak dengan Python.
  • Standard Library mencakup berbagai macam topik dan fungsionalitas, termasuk pemrosesan string, pengelolaan file, pengaturan jaringan, pemrograman GUI, pemrosesan data, matematika, dan banyak lagi.
Standard Library

1. `os` (Operating System Interface):

Digunakan untuk berinteraksi dengan sistem operasi, seperti manipulasi direktori, mengakses variabel lingkungan, dan lainnya.

baca selengkapnya

Memungkinkan penghasilan angka acak.

baca selengkapnya

Untuk bekerja dengan tanggal dan waktu.

baca selengkapnya

tkinter | GUI

Definisi
  • Tkinter adalah toolkit GUI (Graphical User Interface) bawaan yang disertakan dengan Python.
  • Toolkit ini menyediakan alat untuk membuat antarmuka grafis pengguna dengan mudah.
  • Tkinter adalah bagian dari standar pustaka Python dan telah menjadi pilihan populer bagi para pengembang Python untuk membuat aplikasi desktop yang memiliki antarmuka pengguna yang bersahabat.
Tkinter | GUI
Beberapa komponen utama dalam Tkinter termasuk:
  • Widget: Elemen-elemen antarmuka pengguna seperti tombol, label, kotak teks, dan lain-lain.
  • Geometry Managers: Mereka mengatur tata letak widget dalam jendela, seperti pack(), grid(), dan place().
  • Event Handling: Tkinter memungkinkan penanganan peristiwa, seperti klik tombol atau penekanan kunci.
  • Tkinter Classes: Sejumlah kelas di Tkinter, termasuk Tk() untuk membuat jendela utama, dan berbagai kelas widget seperti Button, Label, Entry, dll.

Berikut adalah contoh sederhana penggunaan Tkinter untuk membuat jendela dengan sebuah tombol:

baca selengkapnya
                                                    
    import tkinter as tk

    def on_button_click():
        label.config(text="Hello, Tkinter!")

    # Membuat jendela utama
    root = tk.Tk()
    root.title("Contoh Tkinter")

    # Membuat label
    label = tk.Label(root, text="Selamat datang di Tkinter!")

    # Membuat tombol
    button = tk.Button(root, text="Klik Saya", command=on_button_click)

    # Menata widget menggunakan geometry manager (pack dalam contoh ini)
    label.pack(pady=10)
    button.pack(pady=10)

    # Memulai event loop
    root.mainloop()

Read External File - Open dan With

Definisi
  • Di Python, kita dapat mengakses file eksternal dengan menggunakan fungsi open() dan blok with.
  • Proses membaca file ini dapat dilakukan dengan beberapa mode, seperti "r" (read) untuk membaca file, "w" (write) untuk menulis ke file, dan sebagainya.
Read External File

Di Python, fungsi open() dan blok with digunakan untuk membaca file eksternal dengan cara yang aman dan efisien. Berikut adalah contoh penggunaan open dan with untuk membaca file eksternal:

baca selengkapnya
                                            
    # Membuka file untuk dibaca ("r" mode)
    nama_file = 'contoh.txt'

    # Menggunakan blok 'with' untuk memastikan penutupan file secara otomatis setelah selesai digunakan
    with open(nama_file, 'r') as file:
        # Membaca seluruh konten file sekaligus
        isi_file = file.read()
        print("Isi file:")
        print(isi_file)

    # Setelah keluar dari blok 'with', file akan ditutup secara otomatis

    # Contoh membaca file baris per baris
    with open(nama_file, 'r') as file:
        print("\nMembaca file baris per baris:")
        for baris in file:
            print(baris.strip())  # strip() digunakan untuk menghilangkan karakter newline ("\n") pada setiap baris

    # Contoh membaca sejumlah karakter tertentu
    with open(nama_file, 'r') as file:
        print("\nMembaca 10 karakter pertama:")
        sepuluh_karakter_pertama = file.read(10)
        print(sepuluh_karakter_pertama)
  • Kita dapat membuka file dengan mode "r" (read) untuk membaca isi file.
  • Blok with digunakan untuk memastikan bahwa setelah operasi selesai, file akan ditutup secara otomatis.
  • Dalam contoh pertama, kami menggunakan metode read() untuk membaca seluruh konten file sekaligus.
  • Dalam contoh kedua, kami menggunakan loop for untuk membaca file baris per baris.
  • Dalam contoh ketiga, kami menggunakan metode read(10) untuk membaca 10 karakter pertama dari file.

Penting untuk selalu menangani pengecualian (Exception) yang mungkin terjadi saat membuka atau membaca file, misalnya, untuk menangani file yang tidak ada atau masalah izin akses.

Write External File - Write and Append

Definisi
  • Di Python, Anda dapat menggunakan fungsi open() untuk menulis atau menambahkan konten ke file eksternal.
  • Fungsi ini dapat digunakan dalam dua mode utama: mode "write" untuk menulis ulang file dan mode "append" untuk menambahkan konten pada akhir file.
Write External File

Berikut adalah contoh penggunaan dari Write dan Append:

1. Menulis ke File Eksternal (Write):
baca selengkapnya

  • 'nama_file.txt' adalah nama file yang ingin Anda buat atau timpa.
  • 'w' adalah mode write yang digunakan saat membuka file.
  • Fungsi write() digunakan untuk menulis teks ke dalam file.
  • \n digunakan untuk menambahkan karakter newline (baris baru) antara dua baris teks.
2. Menambahkan ke File Eksternal (Append):
baca selengkapnya

Exception, Error, Try and Except

Definisi
  • Dalam Python, blok try dan except digunakan untuk menangani kesalahan atau pengecualian (exceptions) yang dapat terjadi selama eksekusi program.
  • Pada dasarnya, Anda mencoba menjalankan potongan kode yang mungkin menyebabkan kesalahan, dan jika kesalahan terjadi, Anda menangkapnya dan menangani dengan cara tertentu.
Exception, Error, Try and Except

Berikut adalah struktur umum dari blok try dan except:

baca selengkapnya
                                                    
    try:
        # Potongan kode yang mungkin menyebabkan kesalahan
        # ...
    except ExceptionType1 as e1:
        # Menangani kesalahan dari ExceptionType1
        # ...
    except ExceptionType2 as e2:
        # Menangani kesalahan dari ExceptionType2
        # ...
    else:
        # Blok ini opsional dan akan dijalankan jika tidak ada kesalahan
        # ...
    finally:
        # Blok ini opsional dan akan dijalankan selalu, terlepas dari apakah ada kesalahan atau tidak
        # ...
  • try: Ini adalah blok di mana Anda menempatkan potongan kode yang mungkin menyebabkan kesalahan. Jika kesalahan terjadi di dalam blok ini, eksekusi kode akan beralih ke blok except.
  • except: Blok ini digunakan untuk menangani kesalahan yang spesifik. Anda dapat memiliki beberapa blok except untuk menangani berbagai jenis kesalahan. Setiap blok except memiliki tipe pengecualian (ExceptionType) yang sesuai dengan jenis kesalahan yang ingin Anda tangani. Saat kesalahan terjadi, eksekusi akan melompat ke blok except yang sesuai dengan jenis kesalahan tersebut.
  • as: Ini digunakan untuk menyimpan informasi tentang kesalahan yang terjadi ke dalam variabel. Dengan cara ini, Anda dapat mengakses informasi lebih lanjut tentang kesalahan tersebut.
  • else: Blok ini opsional dan akan dijalankan jika tidak ada kesalahan yang terjadi di dalam blok try. Ini berguna ketika Anda ingin menjalankan beberapa kode hanya jika tidak ada kesalahan.
  • finally: Blok ini opsional dan akan dijalankan selalu, terlepas dari apakah kesalahan terjadi atau tidak. Ini berguna untuk membersihkan sumber daya atau menjalankan kode yang harus dijalankan tanpa peduli apakah ada kesalahan atau tidak.

    try:
        hasil = 10 / 0  # Ini akan menyebabkan ZeroDivisionError
    except ZeroDivisionError as e:

    else:

    finally:
        print("Blok finally akan dijalankan selalu.")

Dalam contoh ini, karena ada kesalahan pembagian dengan nol, eksekusi akan melompat ke blok except, dan pesan kesalahan akan dicetak. Blok else tidak akan dijalankan, tetapi blok finally akan tetap dijalankan.
