Apa Itu HTML

Definisi

Hypertext Markup Language (HTML) adalah bahasa markah standar untuk dokumen yang dirancang untuk ditampilkan di peramban internet. Ini dapat dibantu oleh teknologi seperti Cascading Style Sheet (CSS) dan bahasa scripting seperti JavaScript dan VBScript. Singkatnya HTML adalah bahasa pemrograman yang digunakan untuk merancang sebuah website.

HTML terdiri dari beberapa bagian didalam satu halaman yang ditandai dengan tanda atau </tag/> yang disusun sesuai dengan kebutuhan.

Dalam sebuah website ada beberapa komponen. Dalam pembuatan sebuah website ada beberapa hal dasar yang harus dipelajari, ada HTML, CSS, JavaScript, Pemrograman PHP, dan Database.

Dalam belajar HTML, ada beberapa kebutuhan yang bisa kita gunakan dalam mempelajari HTML, yaitu :

  • Text Editor/Code Editor wadah yang digunakan untuk mengetikan kode-kode HTML dalam menyusun sebuah kerangka website
  • Web Browser untuk menampilkan kerangka website yang sudah kita ketikan di Text Editor
  • Perangkat bisa menggunakan laptop atau PC ataupun smartphone
