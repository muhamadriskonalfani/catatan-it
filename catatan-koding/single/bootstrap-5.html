Bootstrap 5

Definsi

Bootstrap adalah kerangka kerja CSS yang sumber terbuka dan bebas untuk merancang situs web dan aplikasi berbasis web. Kerangka kerja ini berisi templat desain berbasis HTML dan CSS untuk tipografi, formulir, tombol, navigasi, dan komponen antarmuka lainya, serta juga ekstensi opsional JavaScript.

Cara pasang Bootstrap 5
  • Download Bootstrap di browser
  • Buka folder Bootstrap
  • Pilih folder yang berisi folder css dan js
  • Pindahkan folder Bootstrap kedalam folder html
  • Bootstrap 5 siap digunakan
Cara penggunaan Bootstrap 5
CSS

Di dalam tag head masukan tag dibawah ini

<link rel="stylesheet" type="text/css" href="../bootstrap-5.2.3-dist/css/bootstrap.min.css">
JavaScript

Di dalam tag body masukan tag dibawah ini

<scripx type="text/javascript" src="../bootstrap-5.2.3-dist/js/bootstrap.min.js"></script>

Container Bootstrap

Definsi

Container adalah element layout yang paling dasar pada Bootstrap. kita wajib menggunakan container jika ingin menggunakan sistem grid pada Bootstrap. Container bisa kita anggap sebagai wadah yang menampung semua element grid sistem atau element lainnya dari Bootstrap.

Container

Ada dua jenis class container yang bisa digunakan

  • .container

    ukuran lebar layout tidak penuh dan letaknya ditengah

  • .container-fluid

    ukuran lebar layout penuh

Grid Bootstrap

Definsi

Grid sistem adalah sistem pengaturan lebar untuk masing-masing komponen yang ingin kita buat. Pengaturan ini bertujuan untuk mengatur ke-responsive-an website yang kita buat. Jadi kita bisa mengatur grid/kolom sesuai yang kita inginkan dengan sangat mudah. Tanpa harus menulis coding CSS secara manual untuk membuat halaman yang responsive. Ada 12 grid pada Bootstrap. yang bisa kita analogikan sebagai 12 kotak yang bisa kita atur lebarnya dengan mudah.

Grid System

Berikut beberapa jenis ukuran grid system. ukuran-ukuran tersebut ditentukan berdasarkan ukuran layar device.

  • .col-*

    digunakan untuk mengatur grid secara otomatis

  • .col-sm-*

    digunakan untuk mengatur grid pada ukuran layar monitor berukuran kecil (>=576px)

  • .col-md-*

    digunakan untuk mengatur grid pada ukuran layar monitor berukuran sedang (>=720px)

  • .col-lg-*

    digunakan untuk mengatur grid pada ukuran layar monitor berukuran besar (>=992px)

  • .col-xl-*

    digunakan untuk mengatur grid pada ukuran layar monitor berukuran extra besar (>=1200px)

Nb : wajib menggunakan class row jika ingin menggunakan class grid system

Alert, Color & Icon Bootstrap

Definisi

Alert atau semacam pesan pemberitahuan merupakan salah satu komponen penting dalam sebuah aplikasi atau website.

Alert Bootstrap

Ada banyak jenis alert yang disediakan dibootstrap 5, diantaranya adalah

  • alert-success

    Pesan alert ini berwarna hijau, biasanya digunakan untuk notifikasi yang bersifat positif seperti memberitahukan suatu proses telah berhasil dilakukan

  • alert-primary

    Pesan alert primary adalah pesan notifikasi yang berwarna biru

  • alert-secondary

    Pesan alert secondary adalah pesan notifikasi yang berwarna abu-abu

  • alert-danger

    Pesan notifikasi ini berwarna merah, biasanya digunakan untuk memberitahukan proses yang gagal dilakukan

  • alert-warning

    Pesan alert warning adalah pesan yang berwarna kuning

  • alert-info

    Pesan alert info adalah pesan yang berupa informasi dan berwarna biru terang

  • alert-light

    Pesan notifikasi ini berwarna terang, hampir ke abu-abuan

  • alert-dark

    Pesan notifikasi alert dark adalah notifikasi yang berwarna gelap

Icon Bootstrap

Cara menggunakan Icon Bootstrap

  • Masuk ke website bootstrap
  • Pilih docs pada bootstrap
  • Ketikan icon
  • Klik learn more about bootstrap icon
  • Cari icon yang diinginkan
  • Copy link icon font atau svg nya
  • Paste-kan pada tag div disamping text
  • Selesai

Namun jika ingin menggunakanya dengan cara offline maka harus download file nya terlebih dahulu, caranya

  • Kunjungi Bootstrap - Download
  • Kemudian buka link ini
  • Dihalaman webnya silahkan di save
  • Arahkan filenya ke folder bootstrap - js
  • Simpan dengan nama popper.min.js
  • Untuk menggunakanya silahkan masuk ke file HTML
  • Selanjutnya ketikan <scripx type="text/javascript" src="bootstrap/js/popper.min.js"></script>
  • Penempatanya harus diatas script bootstrap.min.js
