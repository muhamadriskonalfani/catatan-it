Menghapus Aplikasi Bawaan

Persiapan

Berikut ada beberapa yang dibutuhkan untuk menghapus aplikasi bawaan, contoh kasus pada HP Xiaomi Redmi5A.

  • PC atau Laptop
  • Kabel data (USB)
  • Aplikasi ADB, jika belum install bisa klik disini
Catatan

Hati-hati untuk menghapus aplikasi bawaan jika Anda tidak tahu konsekuensinya, carilah informasi tentang aplikasi yang akan Anda hapus terlebih dahulu.

Langkah-Langkah

Setelah persiapan selesai, berikut adalah langkah-langkah untuk menghapus aplikasi bawaan.

Pada HP
  • Buka 'Pengaturan' dan masuk ke 'Tentang Ponsel'
  • Lakukan Tap pada 'Versi MIUI' sampai muncul alert 'anda telah jadi pengembang'
  • Masuk ke 'Setelan Tambahan'
  • Aktifkan 'Opsi Pengembang' dan aktifkan 'Debugging USB'
  • Hubungkan HP dengan PC menggunakan Kabel USB
  • Akan muncul notifikasi 'Mengisi Daya', ganti menjadi 'Transfer File'
Pada PC
  • Install ADB
  • Masuk dan Install Device Driver-nya
  • Masuk ke Command Prompt sebagai Administrator
  • Ketikan kode 'adb shell'
  • Pada HP, akan muncul konfirmasi, pilih ok/hubungkan
  • Setelah terhubung Anda bisa menulis perintah yang diinginkan
Kode Fungsi
adb shell untuk menjalankan ADB
pm list packages untuk melihat semua aplikasi yang terpasang di HP
pm uninstall -k --user 0 <nama aplikasi> untuk menghapus aplikasi yang diinginkan
