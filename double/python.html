Mengenal Variabel

Definisi

Variabel adalah sebuah wadah yang bisa menyimpan nilai.

Membuat Variabel

Variabel pada python dituliskan sebagai berikut

x = 10
Penjelasan
  • x adalah nama variabel
  • = adalah assignment atau penugasan
  • 10 adalah nilai yang akan disimpan kedalam variabel x

Pada dasarnya nama variabel pada python dituliskan dengan huruf, tidak boleh menggunakan spasi (bisa gunakan underscore), dan tidak boleh diawali dengan angka.


    nilai_y = 15
    juta10 = 10000000
    nilaiZ = 17.5
Memanggil Variabel

Setelah variabel dibuat kita bisa memanggil variabel tersebut, misalnya kita ingin menampilkan variabel tersebut, cara pemanggilannya adalah sebagai berikut

print(x)

Selanjutnya kita bisa menambahkan teks untuk ditampilkan bersama variabel

print("Nilai x = ", x)
Source Code
                                        
    # Variabel adalah tempat menyimpan data

    # 1. menyimpan / assignment nilai
    a = 10
    x = 5
    panjang = 1000

    # 2. pemanggilan pertama
    print(a)
    print("Nilai x = ", x)
    print("Nilai panjang = ", panjang)

    # 3. penamaan
    # -> tidak boleh pakai spasi, bisa gunakan underscore
    # -> tidak boleh diawali dengan angka
    nilai_y = 15
    juta10 = 10000000
    nilaiZ = 17.5


    # 4. pemanggilan kedua
    print("Nilai a = ", a) # awalnya a adalah 10
    a = 7
    print("Nilai a = ", a) # a berganti jadi 7

    # 5. assignment indirect
    b = a # nilai b diberi nilai sama dengan nilai a
    print("Nilai b = ", b) # hasilnya adalah Nilai b = 7

Tipe Data

Definisi

Tipe data adalah jenis - jenis data yang digunakan untuk variabel pada sebuah bahasa pemrograman.

Jenis Tipe Data Python

Berikut beberapa tipe data dan cara penulisannya pada python

  • Integer

    berisi bilangan bulat (tanpa koma)

    data_integer = 1
  • Float

    berisi bilangan pecahan

    data_float = 1.5
  • String

    berisi kumpulan karakter

    data_string = "ucup"
  • Boolean

    berisi dua nilai yaitu true/false

    data_bool = True

    # tipe data: biner true/false (boolean)
    data_bool = True

Casting Tipe Data

Definisi

Casting tipe data adalah mengkonversi atau merubah tipe data dari satu tipe ke tipe yang lain

Casting Tipe Data
1. Casting Integer

Berikut proses merubah tipe data integer ke float, string, boolean.

baca selengkapnya

    ## INTEGER
    data_int = 9
    print("data = ", data_int, ",type = ", type(data_int))

    data_float  = float(data_int)   # merubah integer ke float
    data_string = str(data_int)     # merubah integer ke string
    data_bool   = bool(data_int)    # merubah integer ke boolean, jika data_int diatas adalah 0 maka boolean nya akan false, jika selain 0 maka true

    print("data = ", data_float, ",type = ", type(data_float))
    print("data = ", data_string, ",type = ", type(data_string))
    print("data = ", data_bool, ",type = ", type(data_bool))
2. Casting Float

Berikut proses merubah tipe data float ke integer, string, boolean.

baca selengkapnya

    ## FLOAT
    data_float = 9.9
    print("data = ", data_float, ",type = ", type(data_float))

    data_int    = int(data_float)     # akan dibulatkan ke bawah, hasilnya 9
    data_string = str(data_float)
    data_bool   = bool(data_float)    # akan false jika nilai float = 0

    print("data = ", data_int, ",type = ", type(data_int))
    print("data = ", data_string, ",type = ", type(data_string))
    print("data = ", data_bool, ",type = ", type(data_bool))
3. Casting Boolean

Berikut proses merubah tipe data boolean ke integer, string, float.

baca selengkapnya

    ## BOOLEAN

    print("data = ", data_bool, ",type = ", type(data_bool))

    data_int    = int(data_bool)     # true akan jadi 1, false akan jadi 0
    data_string = str(data_bool)     # akan jadi teks True
    data_float  = float(data_bool)   # akan jadi 1.0

    print("data = ", data_int, ",type = ", type(data_int))
    print("data = ", data_string, ",type = ", type(data_string))
    print("data = ", data_float, ",type = ", type(data_float))

Input Data Dari User

Definisi

Input pengguna digunakan untuk membuat program dapat meminta pengguna untuk memasukkan teks atau nilai, dan kemudian menggunakan input tersebut dalam eksekusi program. Data yang diinput akan mengembalikan nilai string.

Input User pada Python

Untuk mendapatkan input dari pengguna dalam bahasa pemrograman Python, Anda dapat menggunakan fungsi input(). Berikut adalah contoh sederhana:


    # Meminta input dari pengguna
    nama_pengguna = input("Masukkan nama Anda: ")

    # Menampilkan hasil input
    print("Halo, " + nama_pengguna + "! Selamat datang.")

Perlu diingat bahwa fungsi input() selalu mengembalikan data dalam bentuk string. Jika Anda perlu melakukan operasi matematika atau pemrosesan data numerik lainnya, pastikan untuk mengonversi input ke tipe data yang sesuai. Misalnya, menggunakan int() untuk mengonversi input menjadi bilangan bulat.

Koding
                                        
    # Mengambil input dari user

    # data yang dimasukan pasti string
    data = input("Masukan data: ")
    print("data = ", data, ", tipe = ", type(data))
    
    # jika kita ingin mengambil int, maka
    angka = int(input("Masukan angka: "))
    print("data = ", angka, ", tipe = ", type(angka))
    
    # bagaimana dengan boolean
    biner = bool(int(input("Masukan nilai boolean: ")))
    print("data = ", biner, ", tipe = ", type(biner))

Operasi Aritmatika

Definisi

Operasi aritmatika adalah operasi matematika dasar yang dapat dilakukan pada bilangan untuk melakukan perhitungan. Dalam Python, Anda dapat melakukan operasi aritmatika seperti penambahan, pengurangan, perkalian, pembagian, dan lainnya.

Operasi Aritmatika

Berikut adalah contoh operasi aritmatika dalam Python:

baca selengkapnya

    # Operasi aritmatika

    a = 10
    b = 3

    # operasi tambah + 
    hasil = a + b
    print(a, '+', b, '=', hasil)

    # operasi pengurangan - 
    hasil = a - b
    print(a, '-', b, '=', hasil)

    # operasi perkalian * 
    hasil = a * b
    print(a, '*', b, '=', hasil)

    # operasi pembagian / 
    hasil = a / b
    print(a, '/', b, '=', hasil)

    # operasi eksponen (pangkat) ** 
    hasil = a ** b
    print(a, '**', b, '=', hasil)

    # operasi modulus (sisa bagi) % 
    hasil = a % b
    print(a, '%', b, '=', hasil)

    # operasi floor division (lawan dari modulus, sama dengan pembagian namun dibulatkan kebawah) // 
    hasil = a // b
    print(a, '//', b, '=', hasil)
Prioritas Operasi

Berikut adalah contoh operasi aritmatika dalam Python:

baca selengkapnya

    # Prioritas operasi, operational precedence
    # 1. ()
    # 2. eksponen **
    # 3. perkalian dan teman-teman * / ** % //
    # 4. pertambahan dan pengurangan - +

    x = 3
    y = 2
    z = 4

    hasil = x ** y * (z + x) / y - y % z // x
    print("hasil = ", hasil) # 31.5

    hasil = x + y * z
    print("hasil = ", hasil) # 11

    hasil = (x + y) * z
    print("hasil = ", hasil) # 20

Operasi Komparasi

Definisi

Operasi komparasi adalah operasi yang membandingkan dua nilai atau ekspresi untuk menentukan hubungan antara keduanya. Hasil dari operasi komparasi adalah nilai kebenaran, yaitu True (benar) atau False (salah).

Operasi Komparasi

Dalam Python, terdapat beberapa operator komparasi yang umum digunakan:

1. Sama dengan (==)

Membandingkan apakah nilai x sama dengan nilai y

baca selengkapnya

    x = 5
    y = 5
    hasil = x == y
    print(hasil)  # Output: True
2. Tidak sama dengan (!=)

Membandingkan apakah nilai a tidak sama dengan nilai b

baca selengkapnya

    a = 10
    b = 7
    hasil = a != b
    print(hasil)  # Output: True
3. Kurang dari (<)

Membandingkan apakah nilai m kurang dari n

baca selengkapnya

    m = 8
    n = 12
    hasil = m < n
    print(hasil)  # Output: True
4. kurang dari sama dengan (<=)

Membandingkan apakah nilai p kurang dari sama dengan nilai q

baca selengkapnya

    p = 10
    q = 10
    hasil = p <= q
    print(hasil)  # Output: True
5. Lebih dari (>)

Membandingkan apakah nilai i lebih dari nilai j

baca selengkapnya

    i = 15
    j = 10
    hasil = i > j
    print(hasil)  # Output: True
6. Lebih dari sama dengan (>=)

Membandingkan apakah nilai c lebih dari sama dengan nilai d

baca selengkapnya

    c = 7
    d = 7
    hasil = c >= d
    print(hasil)  # Output: True

Operasi Logika atau Boolean

Definisi

Operasi logika atau Boolean adalah operasi matematika yang bekerja pada nilai True atau False. Dalam pemrograman, operasi logika sering digunakan untuk membuat keputusan berdasarkan kondisi yang diberikan.

Operator Logika

Beberapa operator logika utama yang digunakan dalam Python adalah:

1. Operator `and`

Menghasilkan `true` jika kedua kondisi bernilai `true`.

baca selengkapnya
                                                    
    x = True
    y = False
    hasil = x and y
    print(hasil)  # Output: False
2. Operator `or`

Menghasilkan `true` jika salah satu kondisi atau semua kondisi bernilai `true`.

baca selengkapnya
                                                    
    a = True
    b = False
    hasil = a or b
    print(hasil)  # Output: True
3. Operator `not`

Menghasilkan kebalikan dari nilai kondisi atau ekspresi.

baca selengkapnya
                                                    
    m = True
    hasil = not m
    print(hasil)  # Output: False
4. Operator `xor` (exclusive or)

Menghasilkan `True` jika tepat satu dari dua kondisi atau ekspresi bernilai `True`.

baca selengkapnya
                                                    
    p = True
    q = False
    hasil = p ^ q
    print(hasil)  # Output: True

Operator Bitwise

Definisi

Operator bitwise adalah operator yang bekerja pada level bit dari representasi biner dari nilai-nilai dalam suatu operasi.

Operator Bitwise

Dalam Python, terdapat beberapa operator bitwise yang digunakan untuk melakukan operasi pada bit:

1. AND Bitwise (&)

Menetapkan bit ke-i menjadi 1 jika kedua bit pada posisi ke-i dari operand adalah 1.

baca selengkapnya
                                                    
    a = 5  # Binernya: 0b0101
    b = 3  # Binernya: 0b0011
    hasil_and = a & b
    print(hasil_and)  # Output: 1 (Binernya: 0b0001)
2. OR Bitwise (|)

Menetapkan bit ke-i menjadi 1 jika salah satu atau kedua bit pada posisi ke-i dari operand adalah 1.

baca selengkapnya
                                                    
    x = 5  # Binernya: 0b0101
    y = 3  # Binernya: 0b0011
    hasil_or = x | y
    print(hasil_or)  # Output: 7 (Binernya: 0b0111)
3. XOR Bitwise (^)

Menetapkan bit ke-i menjadi 1 jika hanya satu dari dua bit pada posisi ke-i dari operand adalah 1.

baca selengkapnya
                                                    
    p = 5  # Binernya: 0b0101
    q = 3  # Binernya: 0b0011
    hasil_xor = p ^ q
    print(hasil_xor)  # Output: 6 (Binernya: 0b0110)
4. NOT Bitwise (~)

Menyandi ulang setiap bit, mengubah 0 menjadi 1 dan 1 menjadi 0.

baca selengkapnya
                                                    
    m = 5  # Binernya: 0b0101
    hasil_not = ~m
    print(hasil_not)  # Output: -6 (Binernya: -0b0110)
5. Shift Left (<<)

Memindahkan semua bit operand ke kiri sejumlah langkah tertentu.

baca selengkapnya
                                                    
    c = 4  # Binernya: 0b0100
    hasil_shift_left = c << 1
    print(hasil_shift_left)  # Output: 8 (Binernya: 0b1000)
6. Shift Right (>>)

Memindahkan semua bit operand ke kanan sejumlah langkah tertentu.

baca selengkapnya
                                                    
    d = 8  # Binernya: 0b1000
    hasil_shift_right = d >> 1
    print(hasil_shift_right)  # Output: 4 (Binernya: 0b0100)

Latihan Date and Time

Definisi

Dalam pemrograman Python, modul datetime menyediakan kelas-kelas yang mendukung manipulasi tanggal dan waktu. Modul ini memberikan fungsionalitas yang kaya untuk bekerja dengan tanggal, waktu, dan selang waktu (timedelta).

Latihan Date and Time

Berikut adalah contoh penggunaan modul datetime:

1. Manipulasi Tanggal dan Waktu
                                        
    from datetime import datetime

    # Mendapatkan tanggal dan waktu saat ini
    sekarang = datetime.now()
    print("Tanggal dan Waktu Sekarang:", sekarang)
    
    # Membuat objek datetime dengan tanggal dan waktu tertentu
    tanggal_dan_waktu = datetime(2023, 5, 15, 12, 30, 0)
    print("Tanggal dan Waktu Tertentu:", tanggal_dan_waktu)
    
    # Menampilkan komponen tanggal dan waktu
    print("Tahun:", tanggal_dan_waktu.year)
    print("Bulan:", tanggal_dan_waktu.month)
    print("Hari:", tanggal_dan_waktu.day)
    print("Jam:", tanggal_dan_waktu.hour)
    print("Menit:", tanggal_dan_waktu.minute)
    print("Detik:", tanggal_dan_waktu.second)
2. Format Tanggal dan Waktu sebagai String
                                        
    from datetime import datetime

    sekarang = datetime.now()

    # Menggunakan metode strftime untuk memformat tanggal dan waktu sebagai string
    format_tanggal_waktu = sekarang.strftime("%Y-%m-%d %H:%M:%S")
    print("Format Tanggal dan Waktu:", format_tanggal_waktu)
3. Menghitung Selang Waktu (Timedelta)
                                        
    from datetime import datetime, timedelta

    sekarang = datetime.now()
    
    # Membuat objek timedelta dengan selang waktu tertentu
    selang_waktu = timedelta(days=5, hours=3, minutes=30)
    waktu_setelah_selang = sekarang + selang_waktu
    
    print("Tanggal dan Waktu Sekarang:", sekarang)
    print("Tanggal dan Waktu Setelah Selang Waktu:", waktu_setelah_selang)
4. Parsel Tanggal dari String
                                        
    from datetime import datetime

    tanggal = datetime.strptime(string_tanggal, "%Y-%m-%d")
