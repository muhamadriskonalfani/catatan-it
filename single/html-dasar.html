Apa Itu HTML

Definisi

Hypertext Markup Language (HTML) adalah bahasa markah standar untuk dokumen yang dirancang untuk ditampilkan di peramban internet. Ini dapat dibantu oleh teknologi seperti Cascading Style Sheet (CSS) dan bahasa scripting seperti JavaScript dan VBScript. Singkatnya HTML adalah bahasa pemrograman yang digunakan untuk merancang sebuah website.

HTML terdiri dari beberapa bagian didalam satu halaman yang ditandai dengan tanda atau </tag/> yang disusun sesuai dengan kebutuhan.

Dalam sebuah website ada beberapa komponen. Dalam pembuatan sebuah website ada beberapa hal dasar yang harus dipelajari, ada HTML, CSS, JavaScript, Pemrograman PHP, dan Database.

Dalam belajar HTML, ada beberapa kebutuhan yang bisa kita gunakan dalam mempelajari HTML, yaitu :

  • Text Editor/Code Editor wadah yang digunakan untuk mengetikan kode-kode HTML dalam menyusun sebuah kerangka website
  • Web Browser untuk menampilkan kerangka website yang sudah kita ketikan di Text Editor
  • Perangkat bisa menggunakan laptop atau PC ataupun smartphone

File HTML Pertama

Pendahuluan

Struktur dasar dari HTML adalah sebuah text yang dituliskan dengan kurung lancip. Ada sebuah tag pembuka dan tag penutup, semua yang ada didalam tag pembuka dan tag penutup akan dibaca sebagai file HTML dan akan ditampilkan dibrowser.

Didalam tag HTML ada tag head, di-head ini berisi apa saja yang akan kita gunakan nanti kerangka HTML, atau kebutuhan yang diperlukan dalam menyusun kerangka sebuah website. Selain tag head dalam HTML juga ada tag body, di-body ini adalah semua yang akan ditampilkan di browser.

Hyperlink (Anchor)

Definisi

Hyperlink dapat digunakan untuk menghubungkan halaman 1 dengan halaman yang lain mengingat dalam pembuatan website kita punya beberapa halaman atau file yang saling terhubung untuk memberikan suatu informasi. Tag yang digunakan di HTML untuk Hyperlink adalah Anchor, Anchor ini menghubungkan file induk dengan file tujuan kita. 

<a href="...">...</a>

Jenis Hyperlink

Hyperlink memiliki 2 jenis, yaitu

Hyperlink Internal

  • File berada di folder yang sama

     <a href="nama_file.html"> nama_file </a>

  • File berada di folder dari folder yang sama

     <a href="nama_folder/nama_file.html">nama_file</a>
Hyperlink Eksternal

  • File berada diluar folder

     <a href="../nama_file.html">nama_file</a>

  • Berupa situs website browser

     <a href="www.youtube.com">youtube</a>

Atribut Hyperlink

Ada 2 atribut Hyperlink, yaitu

Atribut "href"

Berfungsi untuk mengarahkan ke halaman lain

 <a href="...">...</a>
Atribut "target"

Berfungsi untuk mengarahkan ke halaman lain dengan membuka tab baru

 <a href="..." target="_blank">...</a>

Video (Format Video)

Definisi

Video (Format Video) digunakan untuk menyisipkan video pada halaman web. untuk menyisipkan video pada halaman web kita membutuhkan tag video. 

                
    <video controls="true">
        <source src="video/black.mp4" type="video/mp4">      
    </video>

Atribut Video

Berikut beberapa atribut dalam format video, yaitu

  • width :
    untuk menentukan lebar

  • height :
    untuk menentukan tinggi

  • controls :
    digunakan untuk menampilkan tombol play, volume, dan sebagainya

  • playsinline :
    digunakan untuk memutar video secara otomatis

  • autoplay :
    juga digunakan untuk memutar video secara otomatis

  • muted :
    digunakan untuk mematikan suara pada video

  • loop :
    digunakan untuk mengulang video tanpa henti

  • poster :
    digunakan untuk menampilkan thumbnail pada video

Tipe Video

Untuk menampilkan video pada halaman web kita harus menyisipkan atribut type pada tag source. Ada 3 type dalam format video, yaitu:

  • video/mp4
  • video/ogg
  • video/webm

Marge Pada Cell Tabel

  • Pilih cell yang akan digabung, Tentukan dibaris dan dikolom keberapa

    • Jika rowspan maka hapus cell td di tr yang berbeda / tr bawahnya
    • Jika colspan maka hapus cell td di tr yang sama
