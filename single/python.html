Mengenal Variabel

Definisi

Variabel adalah sebuah wadah yang bisa menyimpan nilai.

Membuat Variabel

Variabel pada python dituliskan sebagai berikut

x = 10
Penjelasan
  • x adalah nama variabel
  • = adalah assignment atau penugasan
  • 10 adalah nilai yang akan disimpan kedalam variabel x

Pada dasarnya nama variabel pada python dituliskan dengan huruf, tidak boleh menggunakan spasi (bisa gunakan underscore), dan tidak boleh diawali dengan angka.


    nilai_y = 15
    juta10 = 10000000
    nilaiZ = 17.5
Memanggil Variabel

Setelah variabel dibuat kita bisa memanggil variabel tersebut, misalnya kita ingin menampilkan variabel tersebut, cara pemanggilannya adalah sebagai berikut

print(x)

Selanjutnya kita bisa menambahkan teks untuk ditampilkan bersama variabel

print("Nilai x = ", x)

Tipe Data

Definisi

Tipe data adalah jenis - jenis data yang digunakan untuk variabel pada sebuah bahasa pemrograman.

Jenis Tipe Data Python

Berikut beberapa tipe data dan cara penulisannya pada python

  • Integer

    berisi bilangan bulat (tanpa koma)

    data_integer = 1
  • Float

    berisi bilangan pecahan

    data_float = 1.5
  • String

    berisi kumpulan karakter

    data_string = "ucup"
  • Boolean

    # Belajar Casting
    # merubah dari satu tipe ke tipe lain
    # tipe data = int, float, string, bool

    ## INTEGER
    data_int = 9
    print("data = ", data_int, ",type = ", type(data_int))

    data_float  = float(data_int)   # merubah integer ke float
    data_string = str(data_int)     # merubah integer ke string
    data_bool   = bool(data_int)    # merubah integer ke boolean, jika data_int diatas adalah 0 maka boolean nya akan false, jika selain 0 maka true

    print("data = ", data_float, ",type = ", type(data_float))
    print("data = ", data_string, ",type = ", type(data_string))
    print("data = ", data_bool, ",type = ", type(data_bool))


    ## FLOAT
    data_float = 9.9
    print("data = ", data_float, ",type = ", type(data_float))

    data_int    = int(data_float)     # akan dibulatkan ke bawah, hasilnya 9
    data_string = str(data_float)
    data_bool   = bool(data_float)    # akan false jika nilai float = 0

    print("data = ", data_int, ",type = ", type(data_int))
    print("data = ", data_string, ",type = ", type(data_string))
    print("data = ", data_bool, ",type = ", type(data_bool))


    ## BOOLEAN
    data_bool = True
    print("data = ", data_bool, ",type = ", type(data_bool))

    data_int    = int(data_bool)     # true akan jadi 1, false akan jadi 0
    data_string = str(data_bool)     # akan jadi teks True
    data_float  = float(data_bool)   # akan jadi 1.0

    print("data = ", data_int, ",type = ", type(data_int))
    print("data = ", data_string, ",type = ", type(data_string))
    print("data = ", data_float, ",type = ", type(data_float))


    ## STRING
    data_string = "10"
    print("data = ", data_string, ",type = ", type(data_string))

    data_int    = int(data_string)     # akan jadi bilangan bulat 10
    data_float  = float(data_string)   # akan jadi bilangan pecahan 10.0
    data_bool   = bool(data_string)    # akan jadi True jika ada nilanya, akan jadi false jika string kosong ("")

    print("data = ", data_int, ",type = ", type(data_int))
    print("data = ", data_float, ",type = ", type(data_float))
    print("data = ", data_bool, ",type = ", type(data_bool))

    # Prioritas operasi, operational precedence
    # 1. ()
    # 2. eksponen **
    # 3. perkalian dan teman-teman * / ** % //
    # 4. pertambahan dan pengurangan - +

    x = 3
    y = 2
    z = 4

    hasil = x ** y * (z + x) / y - y % z // x
    print("hasil = ", hasil) # 31.5

    hasil = x + y * z
    print("hasil = ", hasil) # 11

    hasil = (x + y) * z
    print("hasil = ", hasil) # 20
